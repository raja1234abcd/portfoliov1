# Full Stack Portfolio Website

A modern full-stack portfolio website with a responsive frontend and a real backend contact form.

## Tech Stack

- HTML5, CSS3, JavaScript
- Node.js, Express
- Netlify (Frontend)
- Render (Backend)

## Features

- Modern dark UI with animations
- Responsive design
- Real contact form connected to backend API
- Deployed and production-ready
